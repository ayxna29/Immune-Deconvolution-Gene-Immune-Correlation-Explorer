# Immune-Deconvolution-Gene-Immune-Correlation-Explorer
Interactive Shiny app for immune cell deconvolution from RNA-seq data. Run quanTIseq, CIBERSORT, TIMER, xCell, EPIC, MCP-counter, ABIS, ConsensusTME &amp; ESTIMATE on TCGA, synthetic, or uploaded expression matrices. Compare cancer types/methods and correlate gene expression with immune cell fractions.
